# Claude CLI Context

This file is automatically loaded as context at the start of each Claude session.

## Project Overview

This is the Claude CLI Learning Project - a comprehensive tutorial for learning Claude CLI features.

## Quick Reference

### Commands
- `/hello` - Greeting with timestamp
- `/count` - File counting utility
- `/weather` - Mock weather command

### Skills
- `/todo` - Task management
- `/commit` - Commit message generator
- `/refactor` - Code refactoring assistant

### Agents
- `@code-reviewer` - Code review specialist
- `@doc-writer` - Documentation writer
- `@test-generator` - Test generation specialist

### Tools
- File Analyzer - Analyze file statistics
- Mock API - Create mock API endpoints
- Code Metrics - Calculate code quality metrics

## Project Structure

```
exercises/
├── 01-custom-commands/    # Slash commands
├── 02-skills/            # Custom skills
├── 03-agents/            # AI agents
├── 04-mcp-tools/         # MCP and custom tools
├── 05-hooks/             # Lifecycle hooks
├── 06-config/            # Configuration
└── 07-integration/       # Integration project
```

## Learning Path

1. Start with custom commands (Module 1)
2. Learn about skills (Module 2)
3. Explore agents (Module 3)
4. Understand MCP and tools (Module 4)
5. Configure hooks (Module 5)
6. Master configuration (Module 6)
7. Complete the integration project (Module 7)

## Best Practices

- Use skills for reusable functionality
- Leverage agents for specialized tasks
- Configure hooks for workflow customization
- Keep tools focused and well-documented
- Use configuration for consistent settings
