# /hello Command

Greet the user with a friendly message and current timestamp.

## Usage
```
/hello
/hello [name]
```

## Output
Displays a greeting message including:
- Personalized greeting with optional name
- Current date and time
- A friendly emoji

## Example
```
User: /hello Alice
Claude: Hello Alice! 👋
       Current time: 2026-03-12 14:30:45
       Welcome to Claude CLI!
```

## Implementation
Respond with a warm greeting that includes:
1. The current timestamp
2. An enthusiastic welcome message
3. A brief mention of what Claude CLI can do
